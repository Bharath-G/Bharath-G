## 📌 Featured Projects

### 🛒 [E-Commerce Microservices Application](https://github.com/Bharath-G/E-commerce-application)
> An event-driven e-commerce backend built with **Spring Boot** and **Apache Kafka**

A full microservice ecosystem simulating a real-world order lifecycle:

| Service | Responsibility |
|---|---|
| **Order Service** | REST API to create & manage orders, publishes `order-created` event |
| **Payment Service** | Consumes order events, processes payments, publishes `payment-success` |
| **Delivery Service** | Schedules delivery, updates order status to `DELIVERED` |
| **Notification Service** | Listens to Kafka topics and triggers alerts |
| **Events Lib** | Shared event contracts across all services |

**Tech:** Java · Spring Boot · Apache Kafka · JPA · Oracle DB · Docker · Lombok

---

### 📝 [DB Logging with OncePerRequestFilter](https://github.com/Bharath-G/dblogging)
> A Spring Boot application demonstrating **request/response logging** to a database using `OncePerRequestFilter`

Intercepts every HTTP request and logs structured data — useful for auditing and debugging in production environments.
